# Mesh V1 Overview

Mesh V1 is our first-generation mesh networking solution, designed to provide reliable and secure communication in various environments.

## Key Features

- **Robust Mesh Networking**: Self-forming and self-healing network topology
- **Secure Communications**: Support for MACsec encryption
- **Flexible Deployment**: Suitable for fixed and mobile installations
- **Simple Configuration**: User-friendly interface for setup and management

## Components

The Mesh V1 system consists of two main components:

1. [**Node**](node.md) - The hardware device that forms the mesh network
2. [**MACsec Configurator**](macsec-configurator.md) - Software tool for configuring secure communications

## Getting Started

To get started with Mesh V1:

1. Review the [Node documentation](node.md) for hardware setup
2. Configure security settings using the [MACsec Configurator](macsec-configurator.md)
3. Deploy your mesh network according to your specific requirements
