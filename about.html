About Natak Mesh

Learn about our company, our mission, and the innovative networking solutions we provide.

Our Story

Natak Mesh was originally inspired by the Rocket M5 MANET builds done by the team at teamawarenesskit.org. We used those solutions for a while but wanted something simpler to set up and use.

Our journey led us to Meshtastic, but the extremely low throughput pushed us back towards WiFi-based solutions. We worked for most of a year on a HaLow WiFi build, but the currently available low-power radios did not have enough of a range advantage to make up for the low throughput.

We built a WiFi mesh core focused on throughput and reliability that can be adapted to different hardware configurations. Two variations are currently in production: the V1.2 with LoRa and Ethernet, and the Mini for compact deployments.

Our Mission & Values

Affordability First

All off-the-shelf consumer grade components. A low-cost entry point into MANET networking using accessible hardware.

Infrastructure Independence

Creates networks without requiring external communications infrastructure. Operates independently in any environment.

Practical Innovation

Leveraging modern technologies like 3D printing for custom case designs, Meshtastic, Reticulum, and other networking protocols to create hardware that offers multiple connectivity options.

Communication Philosophy

Each node provides three communication capabilities that can be used independently or together:

  • IP Mesh Networking: Standard network foundation that supports any networking application
  • Reticulum Integration: Non-IP based communications for privacy-focused applications
  • Standalone Meshtastic: Independent LoRa communication that works with the Meshtastic ecosystem

Use one or all three communication methods as needed. No vendor lock-in.
