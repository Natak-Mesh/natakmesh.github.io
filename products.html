What You Can Do

Connect ATAK Devices

ATAK is designed to work on IP mesh networks like Nucleus provides. The onboard Meshtastic radio can also provide ATAK connectivity via the official ATAK Meshtastic plugin.

Secure Communications

Voice, video, and text over WiFi mesh with Jami. Text, pictures, and voice messages via Reticulum transport. Text and GPS over LoRa using the Meshtastic radio and official app.

Run Reticulum Apps

Reticulum transport runs across the WiFi mesh by default, but is easily expandable to use the internet when available or other connected communication hardware.
