What You Can Do

  • Connect ATAK devices over encrypted WiFi mesh, with optional LoRa radio backup and secure internet bridging via Tailscale
  • Make secure calls with Jami for voice/video/text, or use Meshtastic standalone for long-range text and GPS
  • Run Reticulum apps for encrypted communications over multiple transport methods
  • Extend the platform — the mesh is a full IP network that can host additional services as needed

Nucleus V1.2

Full-featured platform with WiFi mesh, LoRa radio, and Ethernet connectivity.

  • 802.11s WiFi Mesh (WPA3 encrypted, self-healing)
  • WiFi Access Point for device connectivity
