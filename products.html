The Nucleus

Nucleus

ATAK-ready mesh radio with WiFi mesh and LoRa CoT bridge

A portable, battery-powered node running two parallel mesh networks — a high-speed WiFi mesh for full ATAK operations and a long-range LoRa mesh that pushes SA miles beyond WiFi range. Off-grid by default, with an internet gateway when a connection is available.

  • 802.11s WiFi mesh — ~30 Mbps close range, reaching 1–3 miles with elevation and LoS, WPA3 encrypted
  • Built-in Meshtastic LoRa radio — tested 8+ miles with line of sight
  • Native ATAK over WiFi plus a built-in LoRa CoT bridge — no plugins on the phone
  • Optional OpenTAKServer runs right on a node for portable, in-field TAK

Natak TAK Server

Your own private TAKServer, accessible wherever you have a connection

A compact mini-PC that boots straight into a fully configured TAK server. Most teams already have a network connection and just want a TAK server of their own — this gives you one, on more capable hardware than fits inside a Nucleus node. Built-in Tailscale VPN keeps access private, and it can plug into a Nucleus to serve users over the mesh.

  • Official TAKServer — SA, data packages, and mission management
  • Custom web UI for streamlined setup and management
  • Onboard MediaMTX server for low-latency video streaming
  • Built-in Tailscale VPN — private access over any connection, no public cloud
