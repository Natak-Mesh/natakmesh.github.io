The Nucleus

Nucleus

Full-featured mesh node with multiple connectivity options

  • 802.11s WiFi Mesh — ~30 Mbps (~4,400x faster than LoRa). WPA3 encrypted video, voice, and fast data transfer without infrastructure.
  • Built-in Meshtastic LoRa radio — ~6.8 kbps (Short/Fast), tested 8+ miles with LoS
  • Both radios use mesh topology — data hops through every connected node to extend range, automatically rerouting around failures
  • WiFi Access Point for device connectivity
  • Ethernet port (RJ45) for wired connections
  • ATAK works natively over the WiFi mesh — pictures, messaging, position updates, and TAKserver integration at full speed with no additional plugins
  • Onboard OpenDHT server — video, voice, and text via Jami with no internet or central server required
  • Reticulum transport for decentralized encrypted messaging (Sideband, MeshChat)
  • Tailscale VPN for secure remote access when internet is available
  • Optional OpenTAKServer — run a full TAK server directly on a mobile Nucleus node

More About the Platform

🌐

Internet Integration

Plug a node into any internet connection to share it across the mesh and extend secure communications over the existing internet via Reticulum and Tailscale.

🖥️

Web Dashboard

Built-in web interface for monitoring mesh status, configuring node settings, and scanning WiFi channels — accessible from any device connected to the node.

🔩

COTS Components

All off-the-shelf, user-replaceable parts with 3D-printed enclosures. No proprietary hardware — repair or modify with standard components.

🐧

Linux-Based & Extensible

Full Linux OS with a standard IP network. Add services, run scripts, or integrate with existing infrastructure using basic Linux knowledge.
