Nucleus V1

Power it on and you're connected. Nucleus V1 creates an encrypted WiFi mesh between nodes while running dedicated LoRa radios for long-range links. Use ATAK, MeshChat, Sideband, or any app that works on a standard network.

What's Inside

Raspberry Pi computer with external WiFi card and dedicated Meshtastic radio. Everything fits in a 3D printed case with power and ethernet connections. The modular design, commercial off-the-shelf hardware, non-proprietary software, and 3D printed case make repairs, modifications, and upgrades simple.

Radios

  • Mesh WiFi - High-speed links between Nucleus V1 nodes
  • LoRa Radio - Long-range Meshtastic for miles of coverage

What You Can Do

  • ATAK - Share maps, positions, messages, pictures, and voice
  • Reticulum Apps - Use Sideband, MeshChat, and other encrypted messaging
  • Meshtastic - Long-range text messages and position updates using the standard Meshtastic app, or integrate with ATAK via the plugin
  • TAKServer - Optional TAKServer install runs right on the hardware for centralized ATAK coordination and streaming video
  • Standard Apps - Any app that works on a local network
