Nucleus Product Family

Mesh networking nodes for diverse deployment scenarios

Nucleus V1.2

The V1.2 is the full-featured Nucleus platform with WiFi mesh, LoRa radio, and Ethernet connectivity.

Hardware Interfaces

  • 802.11s WiFi Mesh - External radio creates mesh links between nodes
  • Meshtastic LoRa Radio - Onboard radio for long-range communications
  • Ethernet Port - RJ45 connection for wired devices
  • WiFi Access Point - Connect phones, tablets, and laptops to the mesh

Software Capabilities

  • WPA3 Encrypted Mesh - Secure WiFi connections with efficient IP-based routing
  • Mesh Monitoring - Built-in web interface for network management
  • Wi-Fi Channel Analysis - Aircrack-ng tools for spectrum analysis
  • Reticulum Transport - Onboard transport for Reticulum applications like Sideband and MeshChat
  • TAKserver & MediaMTX - Pre-installed but not activated by default

Use Cases

  • ATAK - Works natively over WiFi mesh. Meshtastic radio can be used with the ATAK Meshtastic plugin to fuse LoRa connectivity into ATAK's communication network
  • Reticulum Applications - Run Sideband, MeshChat, or other Reticulum apps over the mesh
  • Meshtastic Messaging - Standalone LoRa node controlled by the Meshtastic app
  • Internet Backhaul - Connect Ethernet port to Starlink or other internet source to provide internet access across the mesh
