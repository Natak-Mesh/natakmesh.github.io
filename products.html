Nucleus V1

Nucleus V1

Nucleus V1 provides off-grid networking through three hardware interfaces:

  • 802.11s WiFi Mesh - External Alfa card creates mesh links between nodes
  • Meshtastic - RAK4631 node for long-range communications
  • Ethernet - Standard RJ45 connection

Software capabilities:

  • IP Connectivity - External WiFi card creates a flat LAN between Nucleus nodes while a bridged Ethernet and local access point allow devices to access the network
  • Reticulum Transport - An onboard transport instance enables devices using Reticulum based applications such as Sideband or MeshChat to find peers on the Nucleus generated local network
  • Meshtastic - Standalone node controlled by the official Meshtastic application on connected devices

What's Inside

Built entirely from COTS (Commercial Off-The-Shelf) and 3D printed components. No custom parts means any component can be replaced readily and cheaply. This approach makes the platform easy to modify for specific requirements.

What You Can Do

  • ATAK - Works natively over the WiFi mesh, will automatically discover and communicate with other ATAK EUDs on the mesh network. Also can utilize the onboard Meshtastic radio for ATAK connectivity using the official Meshtastic ATAK plugin
  • LAN Messaging - Applications such as Jami and Softros can provide communication over the IP network created by the Nucleus node
  • Reticulum - The onboard transport instance allows Reticulum based applications to use the Nucleus' WiFi and Ethernet interfaces to connect to other Reticulum enabled devices on the network independent of IP connections
  • Meshtastic - Onboard node can act as a standalone Meshtastic node using the official Meshtastic application for long range, infrastructureless communication
  • TAKServer - Each Nucleus comes with TAKServer installed along with MediaMTX to enable more ATAK functionality including video streaming. Can be configured as an option from the shop or be configured by the user
