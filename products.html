Mesh V1

Our first-generation mesh networking node with built-in security features, designed for reliable connectivity in any environment.

Key Features

  • Dual-protocol networking with 802.11s WiFi and LoRa radio for extended range
  • WiFi links secured with MACsec link-layer encryption and integrity protection
  • LoRa links protected by end-to-end AES-256 encryption via Reticulum
  • Mesh topology ensures no single point of failure in your network
  • Built with commercial off-the-shelf parts for affordability and repairability
  • Open source software with continuous community-driven improvements
  • Low power consumption allows operation from various 5V sources including USB power banks
  • Compact, durable form factor suitable for various environments
