# Natak Nucleus Node
*Self-healing encrypted mesh network with built-in communications*

## System Capabilities

| System | What It Does | Status |
|--------|--------------|--------|
| **WiFi Mesh** | Encrypted self-healing network backbone—data hops through nodes to extend reach beyond any single radio's range | ✅ Active |
| **Meshtastic/LoRa** | Slow but long-range LoRa-based mesh network for text & GPS | ✅ Active |
| **Jami** | Secure voice/video calls | ✅ Active |
| **Reticulum** | Flexible multi-transport comms | ✅ Active |
| **Tailscale** | VPN to remote infrastructure | ✅ Active |
| **Internet Gateway** | Share internet across mesh | ✅ Active |
| **Web Dashboard** | Monitoring, config, WiFi analysis | ✅ Active |
| **TAKserver + MediaMTX** | ATAK client integration & video streaming infrastructure | ⚙️ On Request |
| **Mumble** | Push-to-talk voice channels | ⚙️ On Request |

## What You Can Do

- **Connect ATAK devices** over encrypted WiFi mesh, with optional LoRa radio backup and secure internet bridging via Tailscale
- **Make secure calls** with Jami for voice/video/text, or use Meshtastic standalone for long-range text and GPS
- **Run Reticulum apps** for encrypted communications over multiple transport methods
- **Extend the platform** — the mesh is a full IP network that can host additional services as needed
